# FrontDesk AI

FrontDesk AI is an intelligent messaging system designed for local service businesses like plumbers, HVAC technicians, and electricians. It handles inbound customer messages, answers common questions using your business profile, collects necessary information, and helps streamline the job booking process so you can focus on doing the work instead of managing your inbox.

## Project Structure

- `/frontdesk-backend` - Node.js + Express API server
- `/frontend` - Next.js + React + Tailwind web application

## Getting Started

### Backend

```bash
cd frontdesk-backend
npm install
npm run dev
```

The backend server will run on `http://localhost:3001`

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The frontend will run on `http://localhost:3000`
